About Orson Charts

Orson Charts is a chart library for the Java(tm) platform that can generate a wide variety of 3D charts for use in applications, applets and servlets. Key features include:
  • multiple chart types: pie charts, bar charts (regular and stacked), line charts, area charts, scatter charts and surface plots;
  • a built-in lightweight 3D rendering engine based on Java2D (no OpenGL or other dependencies, therefore easy deployment);
  • a mouse-enabled chart viewer provides 360 degree rotation and zooming for precise end-user view control;
  • flexible data sources;
  • auto-adaptive axis labeling;
  • support for PDF and SVG export of charts for reporting;
  • a clean and well-documented API, with a high degree of chart configurability;
Please visit http://www.object-refinery.com/orsoncharts/index.html for the latest information about Orson Charts.
